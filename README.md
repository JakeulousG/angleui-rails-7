# Angle UI unofficial conversion to Rails 7
------------

This is a rough port of Angle UI inside Rails 7 using webpacker 4.
Not all of the website theme features are fully tested, so please use at your own risk.

------------


#### Ruby version
**3.0.3**

#### Rails Version
**7.0.2.3**


------------


#####  Configuration
1. `bundle install`
1. `npm install`
1. `db:create`
1. `rails s`
